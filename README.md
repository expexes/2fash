# 2fash

simple 2fa bash client

### Installation

**Dependencies:**
- `oath-toolkit`
- `gnupg`

```sh
git clone https://gitlab.com/expexes/2fash.git
cd 2fash
```



### Usage

**Get help:**
```sh
./2fash help
```

**Create account:**
```sh
./2fash init
```

**Get code:**
```sh
./2fash code [ACCOUNT]
```
